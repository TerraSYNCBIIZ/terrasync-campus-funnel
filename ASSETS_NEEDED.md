## Generation Tips

1. **For Midjourney**: Add `--ar 16:9` for hero images, `--ar 1:1` for portraits, `--v 6` for latest version, `--style raw` for photorealistic
2. **For DALL-E 3**: Specify dimensions in the prompt, add "photorealistic" and "professional photography"
3. **For Ideogram**: Add "photo" tag and specify resolution
4. **Post-processing**: Run through Topaz Gigapixel AI for upscaling, adjust colors in Lightroom to match brand palette
